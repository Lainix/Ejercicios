Nombres

Dado el siguiente arreglo de nombres: arrayNames = ["ada", "grace", "hedy", "margaret", "jean", "ida", "joan"]; Desarrollar el código​ para cada uno de los siguientes casos: Obtener el nombre más largo y escribirlo colocando en mayúscula la primera letra Ordenar el arreglo alfabéticamente en forma ascendente (a-z) y mostrar el resultado en un listado. Importante: no puedes usar los métodos nativos como `sort()`, `map()`, `filter()` o `each()`. Tampoco puedes usar la propiedad `length()` para saber el tamaño de la cadena y tampoco puedes usar un método, ni css para transformar a mayúsculas.
